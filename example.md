# M323 Static Site Generator

Welcome to my **Markdown to HTML** converter built for *Modul 323*.

## Features

- Headings (h1–h6)
- Paragraphs with **bold** and *italic* text
- Unordered lists
- Ordered lists
- Links and images

## Ordered Example

1. Parse Markdown lines into tokens
2. Merge list items recursively
3. Render tokens to HTML
4. Wrap in a full HTML document
